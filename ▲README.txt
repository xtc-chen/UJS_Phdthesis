重点强调：1.需要安装Texlive（如有CTex，需卸载后再装Texlive）
                2.中文只能使用Xelatex编译
                3.文中图片建议采用pdf格式，这样编译比较快（不建议用eps）

本论文模板包含的文件说明：

1、date---------------------------------------------（每一章节的tex文件）
2、ujsthesis.cls------------------------------------- （模板参数文件）
3、reference.bib -----------------------------------（参考文献引用文件）
4、gbt7714-2005-numerical.bst------------------- （GBT7714参考文献格式文件）
5、logo（文件夹）--------------------------------- （此文件夹中存放江苏大学矢量图）
6、figures（文件夹）-------------------------------（此文件夹中存放论文图片）
6、fonts（文件夹）---------------------------------（此文件夹中存放一些字体）
